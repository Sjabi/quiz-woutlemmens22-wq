# 🎯 Quiz Trainer

## Opdracht

Maak een interactieve quizwebpagina met HTML, CSS en JavaScript.

## Wat moet werken?

- Vragen worden getoond
- Antwoorden verschijnen als knoppen
- Klik op een antwoord → feedback tonen
- Volgende vraag knop werkt

## Wat moet jij nog programmeren?

- Vraag tonen
- Antwoorden dynamisch maken
- Antwoord controleren
- Navigeren tussen vragen

## Extra (voor hogere punten)

- Score bijhouden
- Random vragen
- Timer
- Mooie styling

## Tips

- Gebruik `document.getElementById`
- Gebruik lussen (`for`)
- Gebruik functies
